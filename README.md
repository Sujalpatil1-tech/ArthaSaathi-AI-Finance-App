# 💰 ArthaSaathi — AI-Powered Personal Finance Advisor

ArthaSaathi is an AI-based personal finance management web application that helps users track income, manage expenses, analyze financial health, and receive intelligent financial insights and suggestions.

---

## 🚀 Features

- 💰 Income & Expense Tracking  
- 📊 Financial Dashboard (Income, Expenses, Savings)  
- 🧠 AI-Based Financial Insights  
- 💡 Smart Suggestions for Saving & Investment  
- 🤖 AI Chat Assistant for Financial Queries  
- 🔐 User Authentication (Login/Signup)  
- ☁️ Cloud Storage using Supabase  
- 📅 12-Month Financial History Tracking  
- 🌍 Multi-language Support (English, Hindi, Marathi)  

---

## 🎯 Problem Statement

Many individuals struggle to manage their personal finances due to lack of awareness, poor expense tracking, and absence of intelligent guidance. ArthaSaathi solves this problem by providing a simple and AI-driven financial assistant.

---

## 💡 Solution

ArthaSaathi provides a user-friendly platform where users can:
- Track their income and expenses  
- Analyze their financial behavior  
- Receive AI-generated insights  
- Make better financial decisions
